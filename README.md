# Password Strength Checker

This is a simple Python security tool that checks password strength using basic rules.

The script evaluates passwords based on:
- Length of password
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

Purpose:
This project shows basic understanding of password security and Python scripting.

How to run:
1. Install Python
2. Run the script:

python main.py

Example output:
Strong password
Medium password
Weak password
